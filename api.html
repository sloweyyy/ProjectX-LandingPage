Hướng Dẫn Lấy API Key Từ ZaloAI

  1. Truy cập: https://zalo.ai/
  2. Đăng nhập bằng tài khoản Zalo của bạn.
  3. Truy cập: https://zalo.ai/account/manage-keys và chọn 'Create new key'.
  4. Đặt tên gợi nhớ và chọn Text-to-speech.
